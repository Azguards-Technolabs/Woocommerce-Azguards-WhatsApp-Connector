A premium WooCommerce plugin to achieve full feature parity with Magento WhatsApp Connector implementations.

== Description ==

A powerful, enterprise-grade WooCommerce plugin that enables seamless communication between your store and customers via the WhatTack WhatsApp API. Mirroring the feature set of the Magento WhatsApp Connector, this plugin provides robust abandoned cart recovery, shipment tracking, and campaign management.

**External Service Disclosure:**
This plugin relies on the **WhatTack API** (`https://whatatalk-api.azguardstech.com/`) to send WhatsApp messages. You must have an active account with Azguards Technolabs to use this plugin.

== Installation ==

1. Upload the `whatsapp-connector` folder to your `/wp-content/plugins/` directory.
2. Activate the plugin through the 'Plugins' menu in WordPress.
3. Navigate to WooCommerce > Settings > WhatsApp Connector to configure your API credentials.
